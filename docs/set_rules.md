## chunking
- JD는 템플릿·추상 표현·중복 키워드가 많아 그대로 벡터화하면 구분력이 떨어지므로, 반드시 ‘주요 업무·요구 기술·우대 사항·기술 스택·경력 조건’ 같은 의미 단위로 청킹해야 한다.
