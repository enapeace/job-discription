# A (Augmented) — 보강·강화

> **Augmented**: 보강된, 강화된  
> 검색된 정보를 메타데이터와 결합해 LLM에 전달할 컨텍스트를 보강하는 파트입니다.

---

## 검색 결과 + 메타데이터 결합

벡터 검색 결과를 **메타데이터 필터링**과 결합해야 한다.

### 활용 메타데이터

| 필드 | 용도 |
|------|------|
| 회사명 | 특정 회사 JD 필터링 |
| 직무 카테고리 | 인공지능/머신러닝 등 카테고리별 필터 |
| 경력 여부 | 신입/경력 조건 필터 |
| 회사 규모 | 직원 수, 매출 등 기업 규모 기반 필터 |

### 플로우

1. 사용자 질의 → 벡터 유사도 검색
2. 검색된 청크 + 메타데이터 필터 적용
3. 최종 선별된 청크들을 컨텍스트로 묶어 **Generation** 단계로 전달

---

## RAG 구조에서의 위치

```
[R: Retrieval] → [A: Augmented] → [G: Generation]
     ↑                  ↑
  벡터 검색         검색 결과 + 메타 필터 결합
```
