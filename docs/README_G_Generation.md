# G (Generation) — 답변 생성

> **Generation**: 생성  
> 검색·보강된 컨텍스트를 기반으로 LLM이 답변을 생성하는 파트입니다.

---

## 1. 모델 설정

- **OpenAI API**
- 모델: `gpt-4o-mini`

---

사용자 질의 정규화?
기술스택의 경우 첫글자 대문자로 변환하는 tool
시도 시군구 물어보는 경우 sql 검색하는 tool




---

## 2. 예시 질의 (3건)

1. "데이터 파이프라인 구축 경험을 요구하는 JD를 찾아줘."
2. "클라우드 환경에서 서비스 운영 경험을 요구하는 JD는 어떤 게 있나?"
3. "JD 텍스트 기준으로 봤을 때, 데이터 엔지니어와 머신러닝 엔지니어의 경계는 어디에서 갈리는가?"

그 외 **즉석 질의** 지원.

## RAG 구조에서의 위치

```
[R: Retrieval] → [A: Augmented] → [G: Generation]
                                          ↑
                              LLM이 컨텍스트 기반 답변 생성
```
